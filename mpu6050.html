Gyroscope & Accelerometer Sensor

Overview

Used the MPU6050 sensor to collect accelerometer and gyroscope data.

Components

  • Arduino Uno
  • MPU6050
  • Breadboard
  • Jumper Wires

Objectives

  • Measure acceleration
  • Measure rotation
  • Understand I2C communication
  • Visualize sensor data
