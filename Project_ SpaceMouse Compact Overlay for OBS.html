SpaceMouse Compact Overlay

Input Reading• Python • ChatGPT • OBS Studio

Real-time input reading from the 3Dconnexion SpaceMouse Compact to create a dynamic overlay in OBS Studio, for live streaming.

Overview

I started to live stream my blender projects on Twitch, and I wanted to have an overlay that would show my SpaceMouse Compact input in real-time so that the viewers understand better on what I'm doing and how I'm controlling the view. However, when I searched online(github, google, etc.), I couldn't find any existing solution that would allow me to read the input from the SpaceMouse Compact and display it in OBS Studio. So I decided to make my own solution. The goal was simple, to read the input from the SpaceMouse Compact and display it in OBS Studio in real-time. I used Python to read the input from the SpaceMouse Compact and then used ChatGPT to generate a dynamic overlay that would display the input in OBS Studio and fix errors in the python code.
